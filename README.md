# LibertyGSM — 네트워크 검열(SNI/DPI) 우회 학습 프로젝트 (시스템 전체)

**LibertyGSM**은 제한된 네트워크 환경에서 일어나는 DNS·SNI·DPI 검열의 동작 원리를 직접 구현하며 학습하기 위한 유틸리티입니다. **WinDivert**(윈도우 패킷 드라이버)로 트래픽을 드라이버 레벨에서 가로채기 때문에, 프록시 설정이나 브라우저 선택 없이 **시스템의 모든 앱이 자동으로 처리**됩니다.

외부 VPN 서버를 경유하지 않아 **개인정보 유출 우려가 없고, 원래의 네트워크 속도를 최대한 유지**하면서 검열만 우회합니다.

---

## 💡 동작 원리

1. **DNS-over-HTTPS (DoH)**
   - 네트워크의 DNS가 차단 사이트를 엉뚱한 IP로 유도하는 DNS 검열/스푸핑을 막습니다. 나가는 모든 DNS 질의(UDP/53)를 가로채 Cloudflare/Google의 보안 DNS(`1.0.0.1` 등)로 **암호화 해석**해 진짜 IP를 받아옵니다. `1.1.1.1`이 막힌 망을 위해 보조 IP(`1.0.0.1`, `8.8.4.4`)로 자동 폴백합니다.
2. **TLS SNI 레코드 파편화**
   - HTTPS 접속 시 도메인이 평문으로 노출되는 `ClientHello`를 가로채, **여러 개의 유효한 TLS 레코드로 다시 써서**(도메인 문자열 한가운데를 잘라) 전송합니다. DPI 장비는 어느 레코드에서도 차단 도메인 전체를 못 읽지만, 서버는 정상적으로 재조립해 연결합니다. (단순 TCP 분할과 달리, TCP 스트림을 재조립하는 DPI도 우회)
3. **QUIC 차단 → TCP 폴백 유도**
   - 크롬은 차단 사이트에 QUIC(UDP/443, HTTP3)를 먼저 시도하는데 QUIC SNI는 파편화가 불가능합니다. 나가는 UDP/443을 막아 **우리가 파편화하는 TCP/443 경로로 폴백**시킵니다.
4. **완전 투명 동작**
   - 시스템 프록시를 건드리지 않습니다. 프로그램을 끄거나(또는 비정상 종료돼도) 드라이버가 닫히면서 **즉시 원래 네트워크로 복원**됩니다.

## 🚀 실행 방법

> ⚠️ WinDivert가 커널 드라이버를 올리므로 **관리자 권한이 필요**합니다. 실행 시 UAC(관리자 권한 요청) 창이 자동으로 뜹니다.

### 1. Python으로 직접 실행
```bash
pip install -r requirements.txt
python gui.py
```
`gui.py`를 실행하면 관리자 권한으로 자동 재실행됩니다. 창에서 **START**만 누르면 끝입니다. 별도 프록시·브라우저 설정이 전혀 없습니다.

### 2. 단독 실행 파일(.exe)로 빌드
```
build.bat 더블클릭
```
`dist\LibertyGSM.exe`가 생성됩니다. WinDivert 드라이버가 exe 안에 포함되고, 더블클릭하면 관리자 권한을 자동 요청합니다.

## 🛠️ 우회 강도 설정

세 모드 모두 ClientHello를 **유효한 TLS 레코드 여러 개로 재구성**합니다. 실행 중에도 바로 바꿀 수 있습니다.
- **Standard (기본)**: 2개 레코드로 분할(도메인 중앙에서 끊김). 대부분의 차단망에 충분하고 가장 빠릅니다.
- **Advanced (고급)**: 첫 레코드를 아주 작게 + 도메인 중앙 분할로 3개. 정밀한 차단망에 유용.
- **Extreme (강력)**: 약 8개로 잘게 분할 + 조각 사이 지연. 가장 촘촘한 환경용(핸드셰이크가 약간 느려질 수 있음).

---

## 🛠️ 우회 예외(Whitelist) 설정

일부 사이트(예: 국내 게임 로그인 페이지 `nxlogin.nexon.com` 등)나 방화벽/보안 장비가 적용된 서버는 TLS 레코드 파편화를 비정상적인 요청으로 인지하여 연결을 즉시 끊어버릴 수 있습니다.

이러한 도메인을 처리하기 위해 **우회 예외(Whitelist) 설정** 기능을 제공합니다.
- **예외 호스트 설정 파일 (`exclude_hosts.txt`)**: 프로그램이 실행되는 폴더에 생성됩니다. 줄바꿈으로 제외할 도메인을 적어두면 파편화를 적용하지 않고 정상적으로 원본 패킷을 전달해 줍니다.
- **와일드카드 지원**: `*.nexon.com` 형태로 작성하여 서브도메인을 포함해 한 번에 예외 처리를 할 수 있습니다.
- **원클릭 설정**: GUI 화면 내 `🛠️ 제외 도메인 설정` 버튼을 클릭하면 즉시 편집 파일이 실행됩니다.

## ⚠️ 한계

- **IP 기반 차단**은 우회할 수 없습니다(SNI/DNS가 아니라 목적지 IP 자체를 막는 경우). 외부 VPN이 필요합니다.
- 현재 **IPv4만** 처리합니다. 망에서 IPv6가 활성화돼 있으면 일부 트래픽이 우회를 비껴갈 수 있습니다.
- 소속 기관의 네트워크 사용 규정을 반드시 준수하고, 본인 소유 환경에서의 학습·연구 목적으로만 사용하세요.
