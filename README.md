# LibertyGSM - 학교 와이파이 차단 우회 프로그램

**LibertyGSM**은 학교 와이파이나 공공 컴퓨터실 등에서 인위적으로 차단된 웹사이트(웹툰, SNS, 게임, 커뮤니티 등)를 우회하여 자유롭게 접속할 수 있도록 돕는 로컬 프록시 우회 유틸리티입니다.

외부의 유료/무료 VPN 서버를 경유하지 않아 **개인정보 유출 우려가 없으며, 원래의 네트워크 속도를 최대한 유지**하면서 검열을 우회합니다.

---

## 💡 핵심 기능 및 메커니즘

1. **TLS SNI 파편화 (TLS SNI Fragmentation)**
   - HTTPS 웹사이트 접속 시 도메인 정보가 평문으로 노출되는 `ClientHello` 패킷을 미세한 파편으로 쪼개서 전송합니다. 방화벽(DPI 장비)은 패킷이 조각나 있어 차단 대상 사이트인지 알지 못하고 흘려보내지만, 웹사이트 서버는 정상적으로 조각을 재조합하여 연결해 줍니다.
2. **DNS-over-HTTPS (DoH)**
   - 학교 내부 DNS 서버에서 차단 사이트를 잘못된 IP 주소로 유도하는 DNS 검열/스푸핑을 방지합니다. Cloudflare 및 Google의 공식 보안 DNS 서버를 이용해 원래의 정상적인 웹사이트 IP 주소를 받아옵니다.
3. **윈도우 시스템 프록시 자동 제어**
   - 프로그램을 시작하면 윈도우 레지스트리를 설정해 로컬 프록시 서버(`127.0.0.1:10809`)가 작동하게 만듭니다. 프로그램 종료 또는 중지 시 원래의 인터넷 연결 환경으로 완벽하게 복원합니다.
4. **프리미엄 다크 테마 GUI**
   - 세련된 다크 그레이 및 네온 퍼플 디자인의 GUI에서 실시간 접속 도메인과 트래픽 정보를 모니터링할 수 있습니다.

## 🚀 실행 방법

### 1. Python으로 직접 실행하기
Python이 설치되어 있다면 다음 명령어로 간편하게 실행할 수 있습니다 (추가적인 외부 패키지 설치 필요 없음, 100% 표준 라이브러리 사용).
```bash
python gui.py
```

### 2. 단독 실행 파일(.exe)로 빌드해서 사용하기
Python이 없는 다른 컴퓨터에서도 더블클릭만으로 실행하고 싶다면, 동봉된 `build.bat` 스크립트를 실행해 실행파일로 만들어 사용해 보세요.
1. `build.bat` 파일을 더블클릭합니다.
2. 자동으로 `PyInstaller` 패키지를 설치하고 빌드를 진행합니다.
3. 빌드가 완료되면 `dist\` 폴더 아래에 생성된 **`LibertyGSM.exe`** 파일을 가져가서 사용하면 됩니다.

## 🛠️ 우회 강도 설정 안내

우회가 잘 작동하지 않을 때 우회 강도를 조절하여 최적의 상태를 찾을 수 있습니다. 세 모드 모두 ClientHello를 **유효한 TLS 레코드 여러 개로 재구성**(레코드 계층 파편화)하므로, 단순히 TCP 패킷만 쪼개는 방식과 달리 TCP 스트림을 재조립하는 DPI 장비도 첫 레코드에서 SNI를 읽지 못합니다.
- **Standard (기본)**: ClientHello를 2개의 TLS 레코드로 분할 전송합니다(SNI 앞에서 끊김). 대부분의 차단 시스템을 우회하기에 충분하며 속도가 가장 빠릅니다.
- **Advanced (고급)**: 3개의 레코드로 나누고(첫 레코드를 아주 작게) 각 조각 사이에 약 3ms의 딜레이를 줍니다. 패킷 재조합을 시도하는 정밀한 차단망에 유용합니다.
- **Extreme (강력)**: 약 8개의 레코드로 잘게 쪼개고 각 조각 사이에 약 6ms의 시간차를 둡니다. 보안망이 극도로 촘촘한 환경에 유용하지만 최초 핸드셰이크 시 약간의 지연이 발생할 수 있습니다.

## ⚠️ 유의사항

- **파이어폭스(Firefox)** 브라우저는 윈도우 시스템 프록시 설정을 따르지 않고 자체 설정을 사용할 수 있습니다. 파이어폭스를 쓰시는 경우 설정에서 프록시를 `시스템 프록시 설정 사용` 또는 `수동 프록시 설정(HTTP 프록시: 127.0.0.1, 포트: 10809)`으로 맞춰주셔야 작동합니다.
- 본 도구는 교육 목적 및 검열 극복을 위한 용도로 작성되었습니다. 학교/기관의 무단 네트워크 사용 규정을 충분히 숙지하고 이용하시길 바랍니다.
